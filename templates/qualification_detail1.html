CIMEA Fast Verification
In this tool you can check if your certificate is true and compliant with the data that CIMEA has saved in his security systems. You will have three levels of verification: the first one is a manual check between the certificate data and the informations in this page, the second is an automatic check of the PDF file and the third one is the transaction hash created in our blockchain.

Verify your PDF Statement
In this section it is possible to proceed with the manual verification of the PDF Statement, by inserting it in the field below. The Fast Verification system will check that the metadata are correct and identical to the Statement issued by CIMEA in the first place. Please note that here it is possible to insert only the original Statement issued by CIMEA through Diplome.

This section provides access to the Certification Authority that issued the Statement and indicates the date on which the original Statement was published on the blockchain.
